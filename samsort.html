samsort(1)

NAME

samsort — sort SAM and/or BAM files, merging headers where necessary

SYNOPSIS

samsort [-bcm] [-f CMP] [-o FILE] [-S SIZE] [-T DIR] [-z NUM] [FILE]...

DESCRIPTION

It sorts them.

Merging headers

If more than one input file is given, samsort needs to do something about merging headers.

SEE ALSO

The SAM file format specification http://samtools.sourceforge.net/
